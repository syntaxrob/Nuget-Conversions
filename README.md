# Units.Conversions Nuget
This is the project page for the Nuget package **Units.Conversions** which once installed, allows you to convert the following units:

### Length
* Kilometers
* Meters 
* Centimeters
* Millimeters
* Micrometers
* Nanometers
* Miles
* Yards
* Feet
* Inches
* Nautical Miles

### Temperature
* Celcius
* Farenheit
* Kelvin
